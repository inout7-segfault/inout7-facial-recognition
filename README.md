# inout7-facial-rec-dev

## Setup

- Install [`poetry`](https://python-poetry.org/docs/).
- Install requirements:

  ```shell
  poetry install --no-root
  ```

- Activate the environment:

  ```shell
  poetry shell
  ```

## Running

```shell
python3 app.py
```
